# 📊 Project Description

**Project Title:** Personal Productivity Tracker
**Student Name:** Navina M
**Register Number:** 711525BEC074

---

The **Personal Productivity Tracker** is a web-based application designed to help users efficiently manage their daily activities, including tasks, habits, and notes, within a single platform. The main objective of this project is to improve productivity by organizing work and tracking progress in a simple and interactive way.

This application allows users to add, update, and delete tasks while monitoring their completion status through a progress indicator. It also includes a habit tracking system with a streak feature to encourage consistency in daily routines. Additionally, users can create categorized notes with timestamps and pin important information for quick access.

The system is developed using **HTML, CSS, and JavaScript**, ensuring a responsive and user-friendly interface. It also utilizes **localStorage** to store data, allowing users to retain their information even after refreshing the page.

Overall, this project demonstrates key concepts such as **DOM manipulation, event handling, data storage, and responsive design**, making it a practical and effective productivity management tool.
